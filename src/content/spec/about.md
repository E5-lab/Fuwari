# About

::github{repo="E5-lab/Fuwari"}

本站是基于 [Fuwari](https://github.com/saicaca/fuwari) 主题（基于 [Astro](https://astro.build/) 静态站点生成器）搭建的个人博客。

这里不仅是我的个人数字花园，也是我沉淀知识、分享经验的地方。

### 🎨 页面展示与特色

- **极简美观**：遵循现代卡片式与响应式设计，适配手机、平板与桌面端。
- **极致速度**：基于 Astro 驱动，生成纯静态页面，毫秒级加载响应。
- **暗黑模式**：支持深色/浅色/跟随系统的无缝切换。
- **全站搜索**：基于 Pagefind，无需后端接口即可实现高速全站检索。
- **丰富交互**：支持文章分类、标签筛选、阅读时长估计及目录导航。

---

## 🛠️ 深度技术栈 (Tech Stack)

| 模块 | 技术方案 | 描述 |
| :--- | :--- | :--- |
| **核心框架** | [Astro](https://astro.build/) | 专注于内容驱动型网站的高性能静态框架 |
| **主题样式** | [Fuwari](https://github.com/saicaca/fuwari) + [Tailwind CSS](https://tailwindcss.com/) | 灵活且极具美感的博客主题与样式库 |
| **组件开发** | [Svelte](https://svelte.dev/) / [React](https://react.dev/) | 轻量级交互组件支持 |
| **内容解析** | Markdown / MDX | 支持代码高亮、KaTeX 数学公式渲染 |
| **全站检索** | [Pagefind](https://pagefind.app/) | 零配置、离线式静态全站搜索引擎 |
| **托管部署** | [Vercel](https://vercel.com/) / [Netlify](https://www.netlify.com/) | 持续集成 (CI/CD) 自动构建与部署 |
